# Support

## Scoop installation issues

If you're having trouble installing or upgrading chagg via Scoop, [open an issue](https://github.com/codested/scoop-bucket/issues) in this repository.

## Everything else

For bugs, feature requests, and general questions about chagg, use the [main issue tracker](https://github.com/codested/chagg/issues).

- **Documentation:** [chagg.dev/docs](https://chagg.dev/docs)
- **Main repository:** [codested/chagg](https://github.com/codested/chagg)
