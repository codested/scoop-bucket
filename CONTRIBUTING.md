# Contributing

This repository is **automatically managed** by [GoReleaser](https://goreleaser.com) during the [chagg](https://github.com/codested/chagg) release process. The Scoop manifest is generated and pushed automatically — manual changes will be overwritten on the next release.

**There is no need to contribute to this repository directly.**

If you want to contribute to chagg itself, head over to the [main repository](https://github.com/codested/chagg/blob/main/CONTRIBUTING.md).
