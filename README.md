# Scoop Bucket for chagg

This is the official [Scoop](https://scoop.sh) bucket for [chagg](https://github.com/codested/chagg), a release-note workflow tool for Git repositories.

The manifest in this repository is **automatically maintained** by [GoReleaser](https://goreleaser.com) as part of the chagg release process. For documentation, source code, and issue tracking, see the main repository.

**Main repository:** [codested/chagg](https://github.com/codested/chagg)

## Install

```powershell
scoop bucket add codested https://github.com/codested/scoop-bucket
scoop install codested/chagg
```

## Update

```powershell
scoop update chagg
```

## Issues

If you run into installation or upgrade problems specific to Scoop, please [open an issue](https://github.com/codested/scoop-bucket/issues) here.

For bugs, feature requests, or anything related to chagg itself, use the [main issue tracker](https://github.com/codested/chagg/issues).
